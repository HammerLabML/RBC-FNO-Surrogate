## Rayleigh-Bénard Convection
This works seeks to model Rayleigh-Bénard Convection (RBC) using Fourier Neural Operators. RBC describes convection processes in a layer of fluid cooled from the top and heated from the bottom via the partial differential equations:

**Rayleigh-Bénard Convection**
```math
\frac{\partial \bm{u}}{\partial t} + (\bm{u} \cdot \nabla) \bm{u} = -\nabla p + \sqrt{\frac{Pr}{Ra}} \nabla^2 \bm{u} + T \bm{j}\,, \\
\frac{\partial T}{\partial t} + \bm{u} \cdot \nabla T = \frac{1}{\sqrt{Ra Pr}} \nabla^2 T\,, \\
\nabla \cdot \bm{u} = 0
```

The surrogate models are trained on data generated by a Direct Numerical Simulation based on [Shenfun](https://github.com/spectralDNS/shenfun) with following parameters:
| Parameter       | Value |
|-----------------|-------|
| Domain          | ((-1, 1),(0,$2\pi$))      |
| Grid            | 64 x 96              |
| Rayleigh Number | {1e5, 1e6, 2e6, 5e6} |
| Prandtl Number  | 0.7                  |
| ($T_t$, $T_b$)  | (1,2)                |
| $\Delta t$      | 0.025                |
| Episode Length  | 300                  |
| Cook Time       | 200                  |
